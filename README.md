# Homebrew Tap for d-fi

This tap builds and installs d-fi from the upstream Go release tag.

## Install

```sh
brew tap d-fi/tap
brew install d-fi
```

## Usage

```sh
d-fi --help
```

## Formula

- Formula: `Formula/d-fi.rb`
- Version: `2.3.2`
- Source project: <https://github.com/d-fi/GoFi>
- Source tag: <https://github.com/d-fi/GoFi/releases/tag/v2.3.2>
