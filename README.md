# Homebrew Tap for d-fi

This tap installs the prebuilt macOS binary for d-fi.

The d-fi 2.2.0 macOS release asset is an x86_64 binary. On Apple Silicon Macs,
it runs through Rosetta 2.

## Install

```sh
brew tap d-fi/tap
brew install d-fi
```

## Usage

```sh
d-fi --help
```

## Formula

- Formula: `Formula/d-fi.rb`
- Version: `2.2.0`
- Source project: <https://github.com/d-fi/GoFi>
- Binary release: <https://github.com/d-fi/releases/releases/tag/2.2.0>
